# Teogeul (터글)

Korean Keyboard for Android

Based on [hyegyulee/openwnn-korean](https://bitbucket.org/hyegyulee/openwnn-korean/src/master/) and [RivMt/Openwnn-korean-custom](https://github.com/RivMt/Openwnn-korean-custom)

## 기능 Features

하드웨어 키보드와 다양한 배열, 테마를 지원합니다.

Supports hardware keyboard, multiple layouts, and themes. 

### 배열 Layouts

휴대전화와 컴퓨터에서 사용되는 다양한 배열을 지원합니다.

* 두벌식
* 세벌식
* 단모음
* 천지인
* 나랏글

Support below korean keyboard layouts.

* Dubeolsik
* Sebeolsik
* Danmoeum
* Cheonjiin
* Naratgeul

### 테마 Themes

머티리얼 디자인에 기반한 3종과 전자책 리더기를 위한 E-Ink 테마가 제공됩니다.

This app provide 3 themes based on material design and 1 theme for E-Ink devices.

* White
* Dark
* Flat Dark
* E-Ink
