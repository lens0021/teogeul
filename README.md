# Teogeul (터글)

물리 키보드를 위한 한글 입력기.

> [!WARNING]
> 이 리포지토리는 개인 목적을 위해 운영되며 AI Agent가 작성한 커밋이 다수 포함됩니다. 이를 원치 않는 분은 참고하시기 바랍니다.

## 기능 Features

하드웨어 키보드와 다양한 배열을 지원합니다.

Supports hardware keyboard and multiple layouts.

### 배열 Layouts

휴대전화와 컴퓨터에서 사용되는 다양한 배열을 지원합니다.

* 두벌식
* 세벌식
* 단모음
* 천지인
* 나랏글

Support below korean keyboard layouts.

* Dubeolsik
* Sebeolsik
* Danmoeum
* Cheonjiin
* Naratgeul
