# Teogeul (터글)

안드로이드 터치 기기에 물리 키보드를 연결할 때 사용하는 한글 입력기입니다.

> [!WARNING]
> 이 리포지토리는 개인 목적을 위해 운영되며 AI Agent가 작성한 커밋이 다수 포함됩니다. 이를 원치 않는 분은 참고하시기 바랍니다.

## 주요 기능

- 하드웨어 키보드용 한글 입력기
- 한글 자모 조합 방식 선택(표준 자모/첫가끝 조합)
- 모아주기/한 번에 모아치기 옵션 및 대기 시간 설정
- 하드웨어 키보드 한영 키 커스터마이즈

## 지원 키보드 레이아웃

한글

- 두벌식 표준, 두벌식 북한 국규
- 세벌식 390, 세벌식 최종, 세벌식 단모음
- 세벌식 순아래 2014 (안종혁)
- 세벌식 3–2015M (메탈리쟈), 세벌식 3–2015 (소인배), 세벌식 3-P3 (팥알)
- 신세벌식 1995 원안 (신광조), 신세벌식 2003 수정 (박경남), 신세벌식 M (메탈리쟈)
- 모아치기세벌식 안마태 소리글판

영문

- QWERTY, Dvorak, Colemak

## 설치 및 사용

1. 시스템 설정에서 터글 입력기를 활성화합니다.
2. 키보드 전환 UI에서 터글을 선택합니다.
3. 앱 설정 화면에서 하드웨어 키보드 레이아웃을 선택합니다.

## 직접 입력 모드

IME가 `InputConnection`을 얻지 못하거나 엔진 모드가 DIRECT일 때는 직접 입력 모드로 동작합니다.
이 모드에서는 한글 조합 엔진을 거치지 않고, 키 입력을 즉시 문자로 확정해 전송합니다.
